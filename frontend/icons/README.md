# Frontend Placeholder Icons

This directory should contain PWA icons in the following sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## Generate Icons

You can generate these icons from a single source image using online tools:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or use a design tool like:
- Figma
- Adobe Illustrator
- Canva

## Recommended Icon Design

- Simple, recognizable symbol (home with microphone icon)
- Solid background color matching theme (#1a1a2e)
- High contrast for visibility
- Works at small sizes
